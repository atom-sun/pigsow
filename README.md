# pigsow
The love of pig and sow.

```bash
git clone http://github.com/atom-sun/pigsow.git
python3 pigsow/pigsow.py
```

```bash
mv pigsow ~/.pigsow
echo 'alias pigsow="python3 ~/.pigsow/pigsow.py"' >> .bashrc
source ~/.bashrc
pigsow
```
